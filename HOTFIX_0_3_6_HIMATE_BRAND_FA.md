# Hotfix 0.3.8 - HiMate Branding

## تغییر

برند نمایشی نرم‌افزار از «حضور / HiMate Sync» به «HiMate Sync» تغییر کرد.

## مواردی که تغییر کردند

- عنوان پنجره برنامه
- عنوان بالای برنامه
- متن‌های نمایشی مشتری
- نام شورتکات و نام نصب در Inno Setup
- نام نمایشی در About

## مواردی که عمداً تغییر نکردند

- مسیرهای API مثل `/api/hozoor/...`
- نام فایل اصلی سورس `hozoor_customer_app.py`
- نام خروجی فعلی GitHub/Setup برای سازگاری با workflow
- دیتابیس محلی و ساختار داخلی

اگر خواستید در مرحله بعد نام فایل خروجی Setup و exe هم کاملاً HiMate شود، باید workflow و installer هم با هم تغییر کنند.
