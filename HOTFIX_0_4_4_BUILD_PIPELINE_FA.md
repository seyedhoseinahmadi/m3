# اصلاح کامل Build نسخه 0.4.5

در نسخه 0.4.5 مسیر ساخت فایل اجرایی و مسیر ورودی Inno Setup یکپارچه شد.

روند جدید:

1. PyInstaller فایل زیر را می‌سازد:
   `dist\HiMateSync.exe`
2. Workflow وجود فایل را بررسی می‌کند.
3. فایل به این مسیر منتقل می‌شود:
   `Output\HiMateSync.exe`
4. Inno Setup فقط از همین مسیر قطعی استفاده می‌کند.
5. خروجی نهایی:
   `Output\HiMateSync_Setup.exe`

ZIP این نسخه بدون پوشه‌ی اضافه ساخته شده است؛ محتوای ZIP باید مستقیماً در ریشه Repository قرار بگیرد تا پوشه `.github` و فایل‌های برنامه هم‌سطح باشند.
