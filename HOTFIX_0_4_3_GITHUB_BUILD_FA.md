# HiMate Windows 0.4.3 — اصلاح کامل Build گیت‌هاب

این نسخه کامل است و جایگزین نسخه 0.4.2 می‌شود.

اصلاحات اصلی:

- ساخت قطعی فایل `dist\HiMateSync.exe` با نام مورد انتظار Inno Setup
- توقف Workflow با خطای واضح اگر EXE یا Setup ساخته نشود
- حفظ تنظیمات API فهرست کاربران، شعبه‌ها و دستگاه‌ها
- حفظ `AppId` قبلی برای Upgrade صحیح و جلوگیری از نصب دو نسخه جدا
- اصلاح فایل‌های Build محلی و حذف خط اضافی ابتدای فایل‌های BAT
- خروجی نهایی: `HiMateSync_Setup.exe`
