# تغییرات نسخه CUSTOMER-FINAL-INSTALLER-0.2.7

- UI مشتری از حالت خام و تستی خارج شد.
- تب «دستگاه‌ها» به «دستگاه» تغییر کرد.
- صفحه اول بر اساس یک دستگاه متصل طراحی شد.
- نرم‌افزار همچنان به device خاص قفل نیست؛ device_code از خود دستگاه خوانده می‌شود.
- همه مشتری‌ها می‌توانند روی یک سرور مرکزی VPS/ابر آروان باشند و Laravel با device_code تشخیص مشتری/شعبه را انجام می‌دهد.
- رنگ‌بندی بر اساس هدر تیره، پس‌زمینه روشن و تاکید قرمز آوای فردا اصلاح شد.
- پشتیبانی از فونت AFY به‌صورت اختیاری اضافه شد؛ در صورت قرار دادن TTF در assets/fonts، برنامه آن را Load می‌کند.
- Server URL در UI مشتری نیست و هنگام Build داخل EXE بسته می‌شود.
- UTF-8 و Strict ACK حفظ شده است.

خروجی نهایی:

```text
HozoorSyncCustomer_Setup.exe
```

## فونت

پشتیبانی از فونت AFY با مسیر `assets/fonts` اضافه شد. اگر WOFF/WOFF2 مجاز داخل این پوشه باشد، `prepare_fonts.py` قبل از Build تلاش می‌کند TTF بسازد.
