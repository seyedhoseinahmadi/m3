# راهنمای UI نهایی

## تب خانه

برای استفاده روزمره و پشتیبانی سطح اول.

نمایش:

- وضعیت سرور
- تعداد دستگاهی شناسایی‌شده
- رکوردهای در انتظار ارسال
- آخرین همگام‌سازی

دکمه‌ها:

- شروع همگام‌سازی
- توقف
- همگام‌سازی الآن
- بازخوانی

## تب دستگاه

نمایش دستگاهی شناسایی‌شده روی COM/Serial.

## تب همگام‌سازی

نمایش رکوردهای اخیر، وضعیت ارسال و سلامت دیتابیس.

## تب تنظیمات

فقط تنظیمات غیرحساس:

- فاصله خواندن
- فاصله ارسال
- Heartbeat
- Restore
- دریافت دستور
- Port ترجیحی

## تب پشتیبانی

- نسخه
- مسیر دیتابیس
- مسیر لاگ
- خروجی گرفتن از لاگ
- باز کردن پوشه داده
