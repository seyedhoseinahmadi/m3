# Hotfix / Release 0.3.8

## اصلاحات UI

- مسیر دیتابیس، مسیر لاگ و پوشه داخلی از UI مشتری حذف شد.
- صفحه پشتیبانی فقط اطلاعات عمومی و غیرحساس نشان می‌دهد.
- دکمه «بازخوانی» حذف شد؛ بازخوانی UI و سرویس به‌صورت خودکار انجام می‌شود.
- تب‌بندی از حالت خام TTK به ناوبری ساده و قابل ارائه تغییر کرد.
- UI برای یک دستگاه فعال در لحظه اصلاح شد، نه «دستگاه‌ها».
- صفحه خانه فقط موارد پرمصرف را نشان می‌دهد.
- رنگ‌بندی آوای فردا: هدر تیره، تاکید قرمز، کارت‌های روشن.

## اصلاحات فونت

- برنامه می‌تواند فونت AFY را از مسیر زیر لود کند:

```text
assets/fonts/AFYRegular.ttf
assets/fonts/AFYBold.ttf
```

- اگر WOFF/WOFF2 در همین پوشه باشد، `prepare_fonts.py` در Build تلاش می‌کند آن را به TTF تبدیل کند.
- فایل فونت داخل این پکیج قرار داده نشده است؛ اگر مجوز دارید، خودتان در ریپو بگذارید.
