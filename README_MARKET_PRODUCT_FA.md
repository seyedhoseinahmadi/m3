# معرفی فنی محصول برای تحویل

## نام محصول

Hozoor Sync

## نقش نرم‌افزار

نرم‌افزار ویندوزی سبک برای اتصال دستگاه حضور به سرور Hozoor.

## رفتار اصلی

```text
Hozoor Device → USB/Serial → Hozoor Sync → Laravel Server
```

## ویژگی‌ها

- شناسایی خودکار دستگاه
- خواندن device_code از خود سخت‌افزار
- عدم قفل به یک دستگاه خاص
- ذخیره محلی SQLite
- ارسال امن به Laravel
- Strict Safe ACK
- دریافت دستور از سرور
- اجرای دستور روی دستگاه
- ارسال نتیجه دستور به Laravel
- Restore رکوردهای Emergency Offload از سرور
- UTF-8 برای فایل‌ها، لاگ‌ها، JSON و API
- Serial Protocol ساده و ASCII-safe

## UI

طراحی UI طبق استاندارد آوای فردا:

2. سادگی استفاده
