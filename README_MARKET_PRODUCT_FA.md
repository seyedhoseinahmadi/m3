# HiMate Sync - نسخه مشتری

نرم‌افزار ویندوز برای اتصال یک دستگاه HiMate به سرور مرکزی HiMate.

## مسیر اصلی

```text
Device → USB/Serial → HiMate Sync → Laravel Central Server
```

## مدل تجاری

همه مشتری‌ها به یک سرور مرکزی متصل می‌شوند؛ مثلاً VPS روی ابر آروان.  
Laravel با `device_code` تشخیص می‌دهد داده متعلق به کدام مشتری، شعبه یا قرارداد است.

## UI

- ساده
- سریع
- تک‌دستگاه در لحظه
- بدون نمایش اطلاعات حساس داخلی
- مناسب تحویل به مشتری

## تنظیمات حساس

Server URL و Token داخل Build بسته می‌شوند و در UI مشتری نمایش داده نمی‌شوند.
