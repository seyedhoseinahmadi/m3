# Hozoor Sync - نسخه نهایی مشتری

## مدل محصول

همه مشتری‌ها به سرور مرکزی حضور متصل می‌شوند؛ مثلاً VPS روی ابر آروان.

```text
Device → USB/Serial → Hozoor Sync → Central Laravel Server
```

## قانون دستگاه

هر نصب نرم‌افزار در لحظه با یک دستگاه کار می‌کند.

اما نرم‌افزار به یک device خاص قفل نیست. `device_code` از خود دستگاه خوانده می‌شود و Laravel تشخیص می‌دهد این دستگاه متعلق به کدام مشتری/شعبه است.

## UI

- خانه

- همگام‌سازی
- تنظیمات
- پشتیبانی

## تنظیمات حساس

در نسخه مشتری، این موارد داخل UI نیستند:

- Server URL
- Device Code
- Manual Command
- ابزارهای تست زمان

Server URL هنگام Build داخل EXE بسته می‌شود.
