# HiMate Windows — ثبت مستقیم اثر انگشت v0.4.2

این نسخه قابلیت ثبت اثر انگشت را از داخل خود برنامه ویندوز اضافه می‌کند.

## جریان قطعی

1. دریافت فهرست‌ها از:
   - `GET https://mangroup.ir/api/users`
   - `GET https://mangroup.ir/api/branches`
   - `GET https://mangroup.ir/api/devices`
2. انتخاب شعبه، فرد، دستگاه و `finger_id` در تب «ثبت اثر انگشت».
3. کنترل برابری `device_code` انتخاب‌شده با دستگاه فیزیکی متصل.
4. ارسال مستقیم `A {finger_id}` روی Serial با مهلت ۹۰ ثانیه.
5. فقط پس از پاسخ موفق دستگاه، ارسال دقیق این چهار فیلد:

```json
{
  "username": "ali.rezaei",
  "device_id": 1,
  "device_code": "HZ001",
  "finger_id": 8
}
```

به:

`POST https://mangroup.ir/api/fingerprints/registe`

## قوانین

- `user_id` و `event_id` در درخواست ثبت اثر انگشت ارسال نمی‌شوند.
- یکتایی محلی بر پایه `UNIQUE(device_id, finger_id)` است.
- شعبه فقط برای فیلتر رابط کاربری استفاده می‌شود و در درخواست ثبت ارسال نمی‌شود.
- اگر سرور موقتاً در دسترس نباشد، نتیجه موفق سخت‌افزار در SQLite نگهداری و دوباره ارسال می‌شود.
- دستور ثبت اثر انگشت از صف فرمان Laravel اجرا نمی‌شود؛ ثبت فقط از تب ویندوز آغاز می‌شود.
